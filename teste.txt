01. Definir pasta para ser repositório;
02. Executar o Git Bash dentro dela;
03. "git init" para transformá-la em repositório;
04. "git status" para verificar se há arquivos a serem "commitados";
05. "git add nomeArquivo.extensao"
06. "git commit -m "Texto da mensagem do commit"
07. "git remote add origin https://github.com/user/nomeRepositorio.git"
	- conectar repositório a um servidor remoto - 
08. "git push origin master" para enviar ao repositório remoto (github)
	- master pode ser substituído pelo nome de uma nova branch -
09. "git checkout -b leop" - criar uma nova branch -
10. "git add ." para adicionar todos os arquivos do diretório.
